Project Description

The spark for this project came when I first learned about the Google Youtube Data API and it's extensive set of features. Built using ReactJS, this simple web application allows a user to pick any point on the Earth's surface, specify a radius around that point, and find the highest-viewed YouTube video that's been geotagged nearby.

Visit the site and click around!

Technologies Used

  • ReactJS - Frontend design, layout, and functionality
  • Google Maps Javascript API - Map display and user interaction with the map
  • Google YouTube Data API - Locating and linking to video based on the user-provided location
  • Netlify - Remote server hosting
